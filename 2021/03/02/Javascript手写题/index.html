实现一个new

new的具体步骤

  1. 内存中创建一个新对象
  2. 新建对象的 _ proto _指向构造函数的prototype
  3. 调用构造函数，函数的this指向新创建的对象
  4. 执行构造函数内部的代码（给新对象添加属性）
  5. 如果构造函数返回非空对象，则返回该对象，否则，返回新创建的对象
